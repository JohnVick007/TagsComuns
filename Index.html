O HTML5 é a versão mais recente do HTML e suporta novas funcionalidades de linguagem de marcação,
como multimídia, novas tags, elementos e novas APIs. O HTML5 também suporta áudio e vídeo. HTML
não oferece suporte nativo a áudio e vídeo. HTML oferece suporte nativo a áudio e vídeo.

O html é uma linguagem de marcação de texto que consegue conversar com CSS, linguagem
de estilização e com o JavaScript a linguagem de programação,
